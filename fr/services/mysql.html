Bases de données MySQL

Un accès à MariaDb pourra être ajouté, à partir duquel il est possible de créer des bases de données MySQL personnelles. L’identifiant et le mot de passe seront identiques à ceux du compte FTP. Une instance de phpMyAdmin est disponnible à l’adresse suivante : https://club1.fr/phpmyadmin
